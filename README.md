# Bank Loan Report Dashboard

## Overview
This project contains two **interactive Excel dashboards** designed to provide detailed insights into bank loan metrics. The dashboards help stakeholders analyze key financial metrics, loan performance, and customer trends.

---

## Dashboards

### 1. **Summary Dashboard**
The Summary Dashboard provides an overview of key metrics related to loan applications and funding.

#### Key Features:
- **Total Loan Applications**: Displays the total number of applications, Month-to-Date (MTD), and Month-on-Month (MoM) changes.
- **Total Funded Amount**: Highlights the total amount disbursed as loans.
- **Total Loan Amount**: Tracks total loan requests and amounts applied for.
- **Average Interest Rate**: Displays the average interest rate across loans.
- **Debt-to-Income (DTI) Rate**: Shows the ratio of debt to income for applicants.

#### Visuals:
- Loan Applications by **Month**: A line chart showing trends throughout the year.
- Applications by **Term**: Pie chart dividing loans into terms (36 months vs. 60 months).
- Applications by **Purpose**: Breakdown by categories like credit cards, debt consolidation, etc.
- Applications by **Employment Length**: Bar chart showing distribution based on job tenure.
- Applications by **Home Ownership**: Segments such as mortgage, rent, and ownership.

---

### 2. **Loan Performance Dashboard**
The Loan Performance Dashboard focuses on loan quality and deeper financial insights.

#### Key Features:
- **Good Loans vs. Bad Loans Issued**: Pie or bar chart comparing the performance of loans issued.
- **Loan Applications vs. Funded Amount**: Dual-axis chart showing the relationship between applications and disbursed loans.
- **Average Interest Rate vs. Debt-to-Income (DTI) Rate**: Visuals highlighting trends and correlations.

#### Visuals:
- **Good vs. Bad Loans**: A chart classifying loans into successful (good) and defaulted or problematic (bad) categories.
- **Loan Trends**:
  - Bar chart for total loan applications.
  - Trend chart for funded amounts and average interest rates over time.
- **Financial Ratios**: Comparison charts for key financial metrics.

## Tools & Techniques
- **Excel Functions**:
  - Advanced formulas like IF, VLOOKUP, and conditional aggregation.
  - Data validation for accurate entries.
- **Dynamic Charts**:
  - Interactive slicers for filtering data.
  - Conditional formatting for highlighting insights.
- **Visualizations**:
  - Pie, line, and bar charts for better data interpretation.

---

## Applications
This project is valuable for:
- **Loan Managers**: Assessing loan quality and improving loan approval processes.
- **Risk Analysts**: Monitoring debt-to-income ratios and identifying bad loan trends.
- **Executives**: Visualizing financial KPIs for strategic planning.

---

## How to Use
1. Open the **Excel file**.
2. Navigate to the desired dashboard: **Summary** or **Loan Performance**.
3. Use slicers and filters to explore data by:
   - Month
   - Loan Grade
   - Loan Purpose
   - Employment Length
4. Analyze the charts and metrics to draw actionable insights.
